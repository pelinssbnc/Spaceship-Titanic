# 🚀 Uzay Gemisi Titanic (Spaceship Titanic)

Uzay yolcularının anomali sırasında başka bir boyuta
taşınıp taşınmadığını tahmin eden sınıflandırma projesi.

---

## 📌 Genel Bakış

Bu proje **Kaggle Spaceship Titanic** yarışmasını temel almaktadır.
2912 yılında uzay yolculuğu sırasında yaşanan uzay-zaman anomalisinde
hangi yolcuların alternatif boyuta geçtiği tahmin edilmektedir.

Temel hedefler:
- Yolcu verilerindeki eksiklikleri akıllıca doldurmak
- Özellik mühendisliği ile tahmin gücünü artırmak
- Sınıflandırma modellerini doğruluk metriğiyle karşılaştırmak
